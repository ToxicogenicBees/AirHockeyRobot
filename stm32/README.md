# STM32 Software

This subdirectory is in control of an STM32 NEUCLEO-F446RE microcontroller devboard, with additional hardware hat mounted on top. This controls multiple sensors, like ultrasonic distance sensors, limit switches, a temperature sensor, and an emergency stop button, as well as a CoreXY gantry's motors. The gantry runs straight-line motions using Bresenham's line algorithm, the locations of which are provided from the laptop over a USB connection.

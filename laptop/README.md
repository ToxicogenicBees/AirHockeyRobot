# Laptop Software

This subdirectory is in control of capturing images taken by an external camera to find a puck on the table, processing these images to extrapolate a trajectory of the puck, and determining an appropriate response for the gantry. The target responses are send over a USB link to the microcontroller, which then acts on them. The user can control how difficult of an opponent they face, as well as how the puck is tracked, using a custom CLI.

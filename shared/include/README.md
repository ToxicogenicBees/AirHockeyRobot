# Shared Software

This subdirectory contains a few shared files common to both the laptop and microcontroller. These are mostly a few data classes, like Point2, Ray2, and some packet objects. There is also a timer class used to control a variety of things, like timing initialization or controlling serial communication timeouts. Lastly, there is a constants folder containing a variety of useful values.
